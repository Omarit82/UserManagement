# Users Management – Technical Overview

This Android application was developed by Ignacio Carucci and Omar Roselli for TUDAI.

## Authentication

To access the system, users are required to provide valid authentication credentials. In the current preliminary version, the credentials are hardcoded for testing purposes:

** Username: admin@admin.com **
** Password: 1234 **

## Functionality

The application enables users to create and manage a contact list. New contacts can be added through the corresponding interface.

All registered contacts are displayed on the Home screen, where they are automatically sorted in ascending order by last name.

## Search and Filtering

The Home screen includes a search feature that allows users to filter the contact list dynamically. The filtering process is performed based on both first name and last name fields.
